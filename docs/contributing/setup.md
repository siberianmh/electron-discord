# Development Environment Setup

## Requirements

You will need to have installed these tools:

- Node.js v16 or later
- Yarn v1 (legacy, or classic)
- Redis
- MySQL v8

## Verifying

Verify that you install everything run, the output should look similar to below:

```sh
$ node -v
v16.13.0

$ yarn -v
1.22.5
```
